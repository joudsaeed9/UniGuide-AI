# UniGuide-AI

## Overview
UniGuide-AI is an AI-powered platform that helps students choose the most suitable university major and career path based on their interests, skills, and goals.

## Features
- AI-powered major recommendations
- Career path suggestions
- Personalized guidance
- User-friendly interface

## Technologies Used
- Python
- Jupyter Notebook
- Artificial Intelligence
- Machine Learning

## How to Run
1. Clone this repository.
2. Open the notebook in Jupyter Notebook or Google Colab.
3. Run all cells.
